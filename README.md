### Requirements

node >= v16.15.1
npm >= v8.11.0

### Install dependencies

```sh
npm install # Or yarn install
```
###  Server with hot reload at localhost:8080 (PORT loaded from .env file )

```sh
npm  run dev
```

###  Build for production and start server

```sh
npm  start
```
